# HireLens

An Applicant Tracking System built for design hiring managers.

This is a learning project in agentic engineering — built to move from
context-engineering (level 3) to production-grade agentic systems (level 9).

## Status
Week 0 — foundation. No code yet. See `/docs/learning-log/` for weekly progress.

## Stack
- Next.js (App Router) + TypeScript
- Supabase (Postgres + Auth)
- Drizzle ORM
- Anthropic SDK
- Claude Code + Antigravity (dev environment)

## Learning goals
1. Spec-driven development with agents
2. Sub-agent orchestration
3. Custom MCP server building
4. Eval-driven development
5. Production hardening for agentic systems
